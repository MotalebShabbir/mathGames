MERN দিয়ে
বানানো হবে

প্রতিটি layer-এ MERN stack-এর সম্পূর্ণ ব্যবহার হবে। এই একটা project করলেই MERN পুরোপুরি শেখা হবে।

M
MongoDB
Database — সব data store করবে
E
Express.js
Backend — REST API তৈরি করবে
R
React.js
Frontend — সুন্দর UI তৈরি করবে
N
Node.js
Server — সব কিছু চালাবে
+
JWT + Recharts
Authentication + Beautiful Charts

কিভাবে বানাবেন

একসাথে সব না বানিয়ে phase-by-phase এগোন।

PHASE 01 — FOUNDATION
Project Setup + Authentication

Node/Express server, MongoDB connection, JWT login/register, React project setup, protected routes।

Node.js Setup MongoDB Connect JWT Auth React Router Login/Register UI
PHASE 02 — CORE MODULE
Expense Tracker

CRUD operations, category management, Recharts দিয়ে monthly chart। MERN-এর পুরো cycle এখানেই শেখা হবে।

CRUD API Mongoose Schema Form UI Recharts Filter/Search
PHASE 03 — PRODUCTIVITY
Task Manager + Habit Tracker

Task CRUD with priority/deadline, habit daily check-in, streak calculation, heatmap visualization।

Task CRUD Priority System Habit Check-in Streak Logic Heatmap
PHASE 04 — PERSONAL
Journal + Focus Timer + Dashboard

Rich text journal, mood tracking, Pomodoro timer, সব module-এর data একসাথে dashboard-এ দেখানো।

Rich Text Editor Mood System Pomodoro Timer Dashboard Widgets Final Polish
